# 键盘组合键功能指南 / Keyboard Combination Guide

## 中文说明

| 功能分类 | 按键组合 | 功能 |
| :--- | :--- | :--- |
| **Esc功能** | Shift+Esc | ~ |
| | Fn+Esc | ` |
| **Fn功能** | Fn+数字行(1至=) | F行(F1至F12) |
| | Fn+F行(F1至F12) | 数字行(1至=) |
| | Fn+J | Print Screen |
| | Fn+K | Scroll Lock |
| | Fn+L | Pause |
| | Fn+< | Home |
| | Fn+> | End |
| | Fn+Ctrl(右) | Application Key |
